# actions_build_ErfanGSIs  

## 一个基于 Github Actions 制作的自动跑 ErfanGSIs 的脚本。 
### 步骤如下： 
- 首先，fork 这个仓库。
- 然后点击Actions
- 找到Actions_build,设置好参数，就可以运行啦！ 
> 注意：上传为GITHUB发布
